**根因：** Playwright 新版 API 把 `--user-data-dir` 从 `launch()` 参数里移除，改用 `launch_persistent_context(user_data_dir=...)`

- CDP连接策略改为优先直连 9222 端口已有 Chrome

- 改用区块边界法解析（以景区名为边界，固定往后数5行取数值）
