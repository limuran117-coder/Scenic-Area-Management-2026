## 一、项目基本信息

**技术栈：**
- 前端：微信小程序（WXML/WXSS/JS）
- 云端：CloudBase（cloud1-d5gxwed6aa4581e97，上海）
- 云函数：Node.js 18.15

**3个云函数：**
| 函数名 | 用途 | 触发方式 |
|--------|------|---------|
| countdown-sync | 纪念日CRUD + 云端同步 | 手动调用 |
| period-sync | 姨妈记录CRUD | 手动调用 |
| send-reminder | 纪念日到期提醒推送 | 定时触发（每天09:00）|

**已知修复项（需回归验证）：**

- index.js：titleColor缺漏、onShow竞态

- countdownStore.js：upsertLocalItem云端覆盖本地

## 二、测试用例（全部需要执行）

### A1 · 首页纪念日列表（5个用例）

- 首次打开空数据：显示空状态插画+引导
- 有数据时打开：瀑布流卡片正确展示+主题色
- 下拉刷新：数据从云端拉取，列表更新
- 正向/倒数切换：点击「累计时光」数字回拨100天
- 删除：弹确认框→本地+云端同步删除

### A2 · 新建纪念日-add页（8个用例）

- 必填字段验证：不填日期保存→提示「请选择日期」
- 日期选择（未来）：显示「还有X天」
- 日期选择（过去）：自动切换「累计时光」
- 分类选择：选「生日」→ 主题色切换为生日模板
- 封面图上传：拖拽裁切→确认→封面展示裁切图
- 画报生成：详情页→生成纪念画报→产出1080×1440带水印
- 保存成功：跳转首页，新卡片出现
- 编辑保存：编辑后云端更新，首页同步

### A3 · 每日黄历-almanac页（5个用例）

- 黄历数据：显示农历/节气/宜忌/今日一言
- 历史今天：真实显示百度百科历史事件（非虚构）
- 灵签盲盒首次：点击开启→震动+摇签动画+签文
- 灵签重复：当日不再抽，提示「今日已抽取」
- 跨天重置：次日可重新抽取

### A4 · 姨妈记录-period页（4个用例）

- 记录姨妈：输入开始日期+周期→首页出现倒计时卡片
- 姨妈数据同步：记录后刷新首页→卡片正确显示
- 修改姨妈：云端更新，首页同步
- 删除姨妈记录：卡片消失，无残留

### A5 · 云端同步（4个用例）

- 新建后云端存在：新建纪念日→云数据库有记录
- 编辑后云端更新：编辑标题→云数据库已改
- 删除后云端清除：删除→云数据库无残留
- 跨设备同步：手机A新建→手机B打开可见

---

### A6 · 用户授权管理 + 提醒功能（5个用例）⚠️【新增】

- 授权流程：首次打开→微信授权弹框→同意→正常收到推送
- 授权撤回：微信设置→取消授权→小程序内不再收到提醒
- 提醒触达：纪念日到期当天09:00→收到服务通知
- 多用户独立：用户A和用户B各自授权，互不干扰，各自收到各自的提醒
- 免打扰管理：小程序内设「免打扰时段」→该时段不推送

### B · 云函数测试（AI辅助执行）

**B1 countdown-sync（8个用例）：**
- 正常新建 → DB有记录
- 读取列表 → 返回用户全部纪念日
- 更新已有 → DB标题已改
- 删除记录 → DB记录消失
- 空输入容错 → 返回错误，不崩溃
- 无效日期格式 → 正常处理或返回格式错误
- 超长标题 → 截断或正常保存
- 并发写入 → 均成功，无覆盖

**B2 period-sync（5个用例）：**
- 正常记录姨妈 → DB有记录
- 读取姨妈数据 → 返回或null
- 更新姨妈 → DB更新
- 删除姨妈记录 → DB清除
- 多用户隔离 → 用户B不能读用户A数据

**B3 send-reminder定时（4个用例）：**
- 定时触发 → 函数执行
- 有到期纪念日 → 微信服务通知推送
- 空数据 → 不崩溃，不发通知
- 超时保护 → >60秒中断，无脏数据

### C · 数据库安全测试（10个用例）

- 用户A数据 → 用户B无法查询（跨用户隔离）
- OpenId鉴权 → 未登录调用云函数→拒绝
- 跨用户更新 → 用户B更新用户A数据→失败
- 字段缺省保护 → 新建仅传必要字段→云函数补充默认值
- 时间戳正确 → 创建/更新有真实时间戳
- 删除后残留 → 无软删除则物理删除
- 并发一致 → 多设备同时编辑→最后写入胜出
- SQL注入防御 → 标题输入特殊字符→不进DB
- XSS防御 → 前端不执行脚本
- 权限配置 → cloudbaserc.json无公开写权限

---

### D · 压力与异常测试（10个用例）

- 断网新建 → 写入本地队列，恢复后自动同步
- 云函数超时 → 显示加载状态，30秒超时提示重试
- 云函数报错 → 弹toast，不白屏，本地保留
- 空数组防护 → filter/find/map前无数据→不报错
- 日期=今天 → 正确显示「今天」，倒计时为0
- 日期=1970年 → 正常，无负数溢出
- 日期=2099年 → 正常，数字正常
- 标题为空 → 提示必填
- 封面图10MB → 压缩处理，不崩溃
- 列表100条 → 流畅渲染，无卡顿
