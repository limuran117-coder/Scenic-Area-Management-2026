# 竞品深度档案标准格式

> 所有竞品深度分析 wiki 必须严格按此格式归档 | 2026-04-24 制定

---

## 必须包含的7个章节

| 章节 | 内容要求 |
|------|---------|
| **核心数据快照** | 分析日期 + 数据来源 + 关键数字3-5个 |
| **一、百度数据** | 营收/年客量/媒体高频词/知乎标题 |
| **二、[视情况补全]** | 节目表/活动/舆情（如无可跳过） |
| **三、抖音指数分析** | 搜索指数/综合指数/关联词TOP10/人群画像 |
| **四、小红书灵犀数据** | 搜索总量/环比/同比/兴趣榜排名 |
| **五、内容营销规律总结** | 爆款内容类型/角度/情绪/传播路径 |
| **六、对电影小镇的深度借鉴** | 可复制打法/差异化建议（具体，不是套话） |
| **七、关键结论** | 3-5条核心结论，数据支撑 |

---

## 格式规范

### 核心数据快照
```
> 分析日期：YYYY-MM-DD
> 数据来源：抖音创作者平台（CDP）+ 小红书灵犀（CDP）+ 百度搜索（CDP）

| 平台 | 指标 | 数据 | 趋势 |
|------|------|------|------|
| 百度 | 年营收 | XX亿元 | 同比+XX% |
| 百度 | 年游客量 | XX万人次 | 同比+XX% |
| 小红书灵犀 | 搜索总量 | XX万 | 环比+XX% |
```

### 禁止出现
- 「数据暂缺」而不标注来源
- 只有数字没有同比/环比
- 「待补充」等占位符（没有就别写该章节）

### 数据缺失处理
| 情况 | 处理方式 |
|------|---------|
| 平台未返回（关键词太长/太新） | 标注「—」，加括号说明「平台未返回具体数据」 |
| 未采集到 | 标注「未采集到」，不写占位符 |
| 有数据但不确定准确性 | 标注「约XX万」并注明来源 |

## 自动化 vs 人工区分

| 类型 | 触发场景 | wiki标题 |
|------|---------|---------|
| **自动化关键词分析** | 15:00 cron 竞品关键词深度分析触发 | `[关键词]深度分析.md` |
| **人工深度调研** | 按周执行，手动全面采集 | `[景区名]深度分析.md` |

**重要**：
- 人工调研须包含百度营收/游客量/媒体数据
- 自动化分析只包含关键词/人群/灵犀数据，不可替代人工调研
- 两者分别归档，不要混在同一文件
