# 竞品深度分析标准流程

> 项目深度分析标准化操作手册 | 建立：2026-04-20 | 适用：所有竞品分析任务

## 一、四平台数据采集标准流程

### 平台1：百度搜索
**操作：** 直接 `browser.open()` 新Tab导航到搜索结果URL
**URL格式：** `https://www.baidu.com/s?wd={关键词}&rn=20`
**注意：** 不要在已有Tab操作，用新Tab避免验证拦截
**防封：** 每个操作间隔2秒

### 平台2：抖音指数
**操作：** CDP已有直连，导航到关键词分析页
**URL格式：** `https://creator.douyin.com/creator-micro/creator-count/arithmetic-index`
**输入：** 搜索框输入关键词，按Enter
**注意：** 抖音5月10日-7日维护，期间无法访问
**防封：** 每个操作间隔2秒

### 平台3：小红书灵犀（搜索洞察）
**操作：** `page.evaluate()` DOM注入文字触发搜索
**URL格式：** `https://idea.xiaohongshu.com/idea/trend/trendAnalyze`
**注入脚本：**
```javascript
const inp = document.querySelector('input.d-text');
if (inp) { inp.focus(); inp.value = '关键词'; inp.dispatchEvent(new Event('input', {bubbles:true})); }
```
**等待：** 注入后等2秒，再找下拉选项.click()
**原因：** 灵犀搜索框是React组件，普通type/click无效

### 平台4：小红书搜索结果
**操作：** `browser.open()` 新Tab直接开搜索结果页
**URL格式：** `https://www.xiaohongshu.com/search_result?keyword={关键词}`
**注意：** 不用Explore页（会自动播放视频锁死DOM），直接开搜索结果页
**防封：** 每个操作间隔2秒

## 二、数据采集检查清单

- [ ] 百度：营收/游客量/票价/演出场次/媒体报道高频词
- [ ] 抖音指数：搜索指数+综合指数（同比/环比）

## 三、报告标准格式

### 文件命名
`wiki/竞品分析/{竞品名}深度分析.md`

### 必须包含模块
1. 核心数据快照表（四平台汇总）
2. 百度数据（运营/财务/媒体报道）
3. 抖音指数分析（搜索+综合指数）
4. 小红书灵犀数据（搜索量+上下游词）
5. 小红书内容分析（爆款TOP10+热搜问题）
6. 内容营销规律总结（爆款公式/用户决策路径）
7. 对电影小镇的深度借鉴（内容/关键词/定价/运营）
8. 关键结论

### 飞书卡片格式
- 标题：竞品名+核心数据亮点
- 内容：body.elements[]，每个元素tag="markdown"
- 表格：用管道符 `| col1 | col2 |`
- 涨跌：📈/📉
- 异动标注：🔺

## 四、已知问题与解决

| 问题 | 原因 | 解决 |
|------|------|------|
| 百度触发验证 | 在已有Tab重复访问 | 用新Tab open() |
| 灵犀输入框无效 | React自定义input | page.evaluate() DOM注入 |
| 小红书Explore页卡死 | 自动播放视频锁死DOM | 直接开搜索结果页URL |
| 百度web_fetch被封 | 触发了人机验证 | 用browser工具而非web_fetch |
