# 飞书卡片视觉规范

> ⚠️ 2026-04-19 更新：OpenClaw `card` 参数格式与 Feishu API 不兼容，已改用直接 API 调用

---

## 核心原则

- **不出现代码块**（`````` ``` ```````）
- **不降级为纯文本消息**
- **每个模块之间用分隔线区分**
- **emoji作为视觉锚点，引导阅读顺序**

---

## ⚠️ 重要：Feishu 卡片 JSON 格式（直接 API 调用）

> OpenClaw `message(action=send, card=...)` 的 card 参数格式与飞书官方 API 不兼容。
> **必须使用 Python 脚本调用飞书 IM API**，不要用 OpenClaw 的 card 参数。

### 正确格式

```json
{
  "header": {
    "title": {
      "tag": "plain_text",
      "content": "📊 抖音指数日报 | 2026-04-15"
    }
  },
  "elements": [
    {"tag": "markdown", "content": "━━━ 8大景区排名 ━━━"},
    {"tag": "markdown", "content": "**1** | 🌿 清明上河园\\n• 搜索 **174,087** | 📉 -11.46%"},
    {"tag": "markdown", "content": "━━━ 数据亮点 ━━━\\n🔥 建业电影小镇双指标增速第一"}
  ]
}
```

### 关键差异（vs OpenClaw card 参数）

| 字段 | OpenClaw card 参数（错误） | 飞书 API（正确） |
|------|--------------------------|----------------|
| 标题 | `header.title: "字符串"` | `header.title.tag: "plain_text"` + `header.title.content` |
| 元素位置 | `body.elements[]` | 根级 `elements[]` |
| schema | `schema: "2.0"` | 不需要 |

### 发送脚本模板

```python
import json, urllib.request

token = "<获取tenant_access_token>"
chat_id = "oc_2581c03b79e4893cc3616b253d60f34e"

card = {
    "header": {
        "title": {
            "tag": "plain_text",
            "content": "📊 日报 | 2026-04-15"
        }
    },
    "elements": [
        {"tag": "markdown", "content": "━━━ 模块标题 ━━━"},
        {"tag": "markdown", "content": "内容..."}
    ]
}

payload = json.dumps({
    "receive_id": chat_id,
    "msg_type": "interactive",
    "content": json.dumps(card)
}, ensure_ascii=False)

req = urllib.request.Request(
    "https://open.feishu.cn/open-apis/im/v1/messages?receive_id_type=chat_id",
    data=payload.encode("utf-8"),
    headers={
        "Authorization": f"Bearer {token}",
        "Content-Type": "application/json"
    },
    method="POST"
)
with urllib.request.urlopen(req) as resp:
    print(json.loads(resp.read().decode()))
```

## 排版元素库

### 标题分隔线
```
━━━ 关键词 ━━━
```
> 用 `━━━` 而非 `---`，视觉更稳重

### 模块编号
```
🌿 01 | 标题文字
```
> emoji + 两位数编号 + `|` 分隔

### 要点列表
```
• 关键内容
• 关键内容（加粗：**高亮词**）
```

### 高亮数字
```
增长 **+68.6%**（数字加粗突出）
```

## 视觉分级规则

| 级别 | 符号 | 用途 |
|------|------|------|
| 🔥 | 爆点/异动 | 重要异动、核心数据 |
| 📈📉➖ | 涨跌状态 | 趋势标注 |
| ⚠️ | 警示 | 风险、问题 |
| 💡 | 洞察 | 建议、分析 |
| 🎬🌿🌍📺 | 模块标识 | 不同类型内容区分 |
