# 案例库更新 SOP

> 每日20:00从全国景区爆款内容中筛选优质案例，归档进入案例库 | 仅更新wiki，不发群

---

## 归档标准

内容有创意、或流量异常高、或营销手法新颖，三者满足其一即可入库。

## 执行步骤

1. 扫描抖音/小红书/微博/B站等平台的景区爆款内容
2. 筛选具有营销方法论价值的案例（不只是流量高，要有借鉴意义）
3. 按模板归档到 `wiki/案例库/`（标题/平台/数据/爆款分析/借鉴思路/标签）
4. 更新案例库索引文件

## 归档模板

```markdown
# [案例标题]

- **平台**：
- **数据**：点赞/收藏/转发
- **爆款分析**：为什么爆
- **借鉴思路**：电影小镇可怎么用
- **标签**：#景区 #营销 #...
```

## 本任务不发送到群

案例库更新只写入 wiki，不发送飞书卡片到群。
