# 每日复盘整合 SOP

> 每日17:00对当天所有任务执行结果进行汇总复盘 | 每日17:00发送

---

## 汇总内容（必须包含4个部分）

1. **今日各任务执行情况**：成功/失败/异常
2. **关键数据发现**：抖音指数/竞品动态/行业热点中的重要信号
3. **潜在问题**：数据缺失/平台异常/执行遗漏
4. **次日改进建议**

## 报告格式

```json
{
  "schema": "2.0",
  "header": {
    "title": {"tag": "plain_text", "content": "📋 每日复盘整合 | YYYY-MM-DD"},
    "template": "purple"
  },
  "body": {
    "elements": [
      {"tag": "markdown", "content": "## 📌 一、任务执行情况\n\n✅ 成功任务：...\n⚠️ 异常任务：...\n❌ 失败任务：..."},
      {"tag": "markdown", "content": "## 📌 二、关键数据发现\n\n• [发现1]\n• [发现2]..."},
      {"tag": "markdown", "content": "## 📌 三、潜在问题\n\n• [问题1]\n• [问题2]..."},
      {"tag": "markdown", "content": "## 📌 四、次日改进建议\n\n① [建议1]\n② [建议2]..."}
    ]
  }
}
```
