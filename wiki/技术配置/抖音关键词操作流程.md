## 完整操作流程

### Step 1: 进入关键词分析页
```
URL: https://creator.douyin.com/creator-micro/creator-count/arithmetic-index
```

### Step 2: 搜索关键词
1. 在页面搜索框输入当日关键词
2. 按Enter或点击搜索按钮
3. 等待页面加载（3-5秒）

### Step 3: 抓取三类数据

#### 数据1：关联词（搜索关联词TOP10）
- 位置：页面「关联词」区域
- 数据：关键词名称 + 热度指数
- 用途：了解用户搜索路径

#### 数据2：人群画像
- 位置：页面「人群画像」区域
- 数据：
  - 性别分布（男/女）
  - 年龄分布（18-23/24-30/31-35/36-40/41+）
  - 地域分布（TGI排名）
- 用途：精准投放定向

#### 数据3：关键词数据（内容分布）
- 位置：页面「关键词」或「内容」区域
- 数据：
  - 搜索指数
  - 综合指数
  - 内容分/搜索分/传播分
  - 视频量/点赞量

---

## 数据字段说明

### 关联词
| 字段 | 说明 |
|------|------|
| 关联词 | 与主关键词相关的搜索词 |
| 热度指数 | 该关联词的搜索热度 |

### 人群画像
| 字段 | 说明 |
|------|------|
| 性别分布 | 男性/女性占比 |
| 年龄分布 | 各年龄段占比 |
| 地域TGI | 地域热度指数（河南通常TGI最高） |

### 关键词数据
| 字段 | 说明 |
|------|------|
| 搜索指数 | 用户主动搜索的热度 |
| 综合指数 | 内容+互动+搜索的综合热度 |
| 内容分 | 内容创作热度 |
| 搜索分 | 搜索热度 |
| 传播分 | 内容传播力度 |

---

## 注意事项
- 页面数据下午更新，08:00采集的是昨日数据
- 综合指数排序由高到低
- 环比>±20%用🔺标注
