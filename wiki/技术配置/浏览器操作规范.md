## 2. 小红书灵犀后台 · 搜索洞察（关键！）

**URL：** `https://idea.xiaohongshu.com/idea/trend/trendAnalyze`

### 操作步骤

**Step 1：** 导航到灵犀页面
```
browser.navigate(url=灵犀URL, targetId=已有Tab的targetId)
```

**Step 2：** 等待页面加载
```
等待2秒：sleep 2
```

**Step 3：** DOM注入文字（❌不要用type()/click()）
```
browser.act(targetId=xxx, kind="evaluate", fn=() => {
  const inp = document.querySelector('input.d-text');
  if (inp) { inp.focus(); inp.value = '关键词'; inp.dispatchEvent(new Event('input', {bubbles:true})); }
  return inp ? 'OK' : 'not found';
})
```

**Step 4：** 等下拉出现
```
sleep 2
```

**Step 5：** 找下拉选项并点击
```
browser.act(targetId=xxx, kind="evaluate", fn=() => {
  const opts = document.querySelectorAll('.d-grid.d-option.d-option-content');
  for (const o of opts) {
    if (o.textContent.trim() === '万岁山武侠城') { o.click(); return 'clicked'; }
  }
  return 'not found, total: ' + opts.length;
})
```

**Step 6：** 等待数据加载
```
sleep 3
```

**Step 7：** snapshot获取数据
```
browser.snapshot(timeoutMs=20000)
```

### 找选项的技巧

下拉列表项用class选择器 `.d-grid.d-option.d-option-content`，textContent包含关键词。
**精确匹配**要找 `o.textContent.trim() === '关键词'`，避免点错（比如搜"万岁山武侠城"要选包含这个完整词的选项）。

### 其他模块路径

- 搜索洞察：`/idea/trend/searchInsight`（需权限，可能404）
- 笔记洞察：`/idea/trend/noteInsight`
- 趋势榜单：`/idea/trend/trendRank`

## 3. 抖音关键词指数 · 操作规范

**URL：** `https://creator.douyin.com/creator-micro/creator-count/arithmetic-index`

### 操作步骤

**Step 1：** 导航到关键词页（用已有Tab或开新Tab）

**Step 2：** 在搜索框输入关键词
```
browser.act(targetId=xxx, kind="type", ref="搜索框ref", text="万岁山武侠城")
```
**或用：** `browser.act(..., kind="click", ref="搜索框ref")` → 再 `kind="type"`

**Step 3：** 按Enter
```
browser.act(targetId=xxx, kind="press", key="Enter")
```

**Step 4：** 等待数据加载
```
sleep 3
```

**Step 5：** snapshot获取数据

## 4. 百度搜索 · 操作规范

**URL格式：** `https://www.baidu.com/s?wd={关键词}&rn=20`

### 操作步骤

**必须：** `browser.open()` 开新Tab（禁止在已有Tab刷新！）
```
browser.open(url="百度搜索URL")
```

**防封：** 等待4秒再做snapshot，每个操作间隔2秒。

### 采集内容

- 搜索结果标题+摘要
- 相关搜索词
- 热搜榜
- 百度笔记Tab内容

## 5. 小红书搜索结果页 · 操作规范

**URL格式：** `https://www.xiaohongshu.com/search_result?keyword={关键词}`

### 操作步骤

**必须：** `browser.open()` 开新Tab（直接开搜索结果URL，绕过Explore页）
```
browser.open(url="小红书搜索结果URL")
```

**防封：** 等待5秒做snapshot，每个操作间隔2秒。

### 采集内容

- 爆款笔记标题+点赞量+作者
- 热搜问题（"大家都在搜"）
- 标签筛选器内容
- 不同筛选Tab（综合/今日现状/必看节目等）

### 筛选按钮操作

用 `browser.act(..., kind="evaluate", fn=() => { /* 找按钮并.click() */ })`
按钮textContent示例：`'今日现状'`、`'必看节目'`、`'官方门票'`

## 6. CDP连接不稳定处理

**症状：** `Error: timed out` / `Frame has been detached`

**处理：**

| 场景 | 操作 |
|------|------|
| 单个Tab崩溃 | `browser.open()` 开新Tab继续 |
| 多个Tab失效 | `browser.tabs` 列出所有，用正常Tab的targetId |
| 所有Tab崩 | `browser.stop()` 再 `browser.start()` |

**预防原则：**
- 每次操作间隔2秒（`sleep 2`）
- 数据采集用新Tab，不复用已有Tab
- 避免连续快速点击
