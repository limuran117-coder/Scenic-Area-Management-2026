## 完整操作流程

### Step 1: 进入趋势分析页
```
URL: https://idea.xiaohongshu.com/idea/trend/trendAnalyze
```

### Step 2: 搜索关键词
1. **激活搜索框（用JS）：**
   ```javascript
   const inputs = document.querySelectorAll('input');
   for (let inp of inputs) {
       if (inp.offsetParent !== null) { inp.click(); inp.focus(); break; }
   }
   ```
2. **输入关键词（用keyboard.type，慢速）：**
   ```javascript
   await page.keyboard.type("万岁山武侠城", delay=100)
   ```
   - 每个字符延迟100ms，模拟真人输入
3. **等待下拉建议出现（约3-4秒）**
4. **按Enter选择精确匹配** ← 关键！不要点击下拉词，会导致关键词带后缀

## 技术要点

### 为什么要用JS点击输入框？
页面输入框可能是隐藏的或opacity:0，常规click会报"element not visible"

### 为什么要用keyboard.type而不是fill？
fill速度太快，会被平台识别为机器操作

### 为什么要按Enter而不是click下拉选项？
click会选择带后缀的词（如"万岁山武侠城门票"），Enter才选精确匹配词

### 内容Tab的JS定位方法
```javascript
// 找包含"内容"且父元素同时有"搜索"和"内容"的元素
for (let el of allElements) {
    if (el.innerText?.trim() === '内容') {
        const parent = el.parentElement;
        if (parent) {
            const children = Array.from(parent.querySelectorAll('*')).map(e => e.innerText?.trim());
            if (children.includes('搜索') && children.includes('内容')) {
                el.click();
                break;
            }
        }
    }
}
```

## 数据字段说明

### 搜索Tab
| 字段 | 说明 |
|------|------|
| 搜索总量 | 该关键词总搜索次数 |
| 环比 | 较上周变化 |
| 同比 | 较去年同期变化 |
| 波峰 | 历史最高点 |
| 热门搜索词 | 用户搜索的高频词TOP10 |
| 搜索上下游词 | 用户搜索路径 |

### 内容Tab
| 字段 | 说明 |
|------|------|
| 阅读总量 | 该关键词下所有笔记的总阅读量 |
| 环比/同比 | 阅读量变化趋势 |
| 笔记关键词分布 | 笔记中提到的高频词及笔记数 |
| 相关热门笔记 | 该关键词下的高阅读量笔记 |
